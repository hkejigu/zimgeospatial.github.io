    Welcome

    Geospatial data on Zimbabwe is largely scattered all over the web and efforts by the national data custodians to implement distribution infrastructure/ initiatives are minimal if existent. One usually finds this data embedded in pdfs, spreadsheets or images. Making any use of the same requires converting to the appropriate format of which time one may not have.Additionally there is lack of contextual exposure of data resulting in stifled innovative use of this data. Try finding Murerekwa Primary School for instance on an online map..[now that you're back, from GeoNames or OpenStreetMap, if you tried hard enough), now, read on!

    Which Zimbabwe?

    In case you're here by accident but would stay, Zimbabwe is a nation in Africa. The map below should (hopefully) help you get oriented.

    Project Motivation

    • Pool together usable geospatial data on Zimbabwe in a somewhat 'quickly' usable format from whichever source inorder to promote innovative use of data.

    Repository Structure

    Each repository contains a set of files which are intended to serve varying purposes. The list below gives an insight into each file's contents and what purpose each file serves. Notice that there are similar file names but different file extensions.

    • README.md :: Contains the metadata about the repository. To be used for checking on data currency and attribution.


    • filename.geojson :: For geographic view of data :: Is the spatial data. To be used for previewing the data within the github repository and for manipulation with tools such as GeoJson.io - a quick, simple tool for creating, viewing, and sharing maps and GitSpatial - a spatial API that fronts and syncs GeoJSON hosted by GitHub. The popular QGIS can also be used to further exploit the data.


    • filename.zip :: For data manipulation in GIS software:: This is the spatial data in zipped (compressed) ESRI shapefile. The shapefile is supported natively by many GIS softwares. QGIS can even open zip file for viewing purposes without the need to decompress its contents first.


    • filename.csv :: For tabular view of data:: This format is useful for searching using the GitHub CSV and TSV search functionality say you just want to find out if an entry you are interested in exists in the dataset.

    Peruse The Data

    • 1. The Catalog

      If you want to quickly peruse the entire dataset collated for this project visit The Searcheable Data Catalog

    • 2. The Map

      Kwedu is a map product that uses data hosted on this project's data repositories. From there you can search for place names, schools, hospitals and other such information and the map will zoom to that place to povide contextual information. Access Kwedu

    Participating

    You're welcome to consume and edit datasets. We're particularly interested in having up-to-date data and correct attributes, spellings e.t.c. Be sure to feedback via email (zimgeospatial [at] gmail.com] or twitter @zimgeospatial

    • 1. If you're geeky, folk, edit, submit pull request.
    • 2. If you've no idea what 1 was about, just download the zipped shapefile and you are good to go.
    • 3. Or simply email the data you wish to share or intend to receive.
